# How install

Install Hyperbee, Hyperdrive and hyperswarm from HyperCore

```
npm install hyperbee hypercore hyperdrive hyperswarm
```
Run local server

```
 http://localhost:3000/
```

Get pair 
```
curl http://localhost:3000/pairs
```
